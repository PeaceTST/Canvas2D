Include-Flags für XUnPack (XUP::)
Alle Flags können kombiniert werden - #IsC2D_XUnPack = #XUP_PACK | #XUP_APLB | #XUP_JCG1 | ...

#IsC2D_XUnPackBeschreibung
#XUP_APLBEntpacken von aPLib
#XUP_BLZ1Entpacken von BriefLZ (PB-Native)
#XUP_BZP2Entpacken von BZip2 & #XUP_XPKF BZP2
#XUP_CALLZusätzlicher Callback für #XUP_PACK (#XUP_APLB, #XUP_JCG1) während dem Packvorgang
#XUP_FILE*Memory = @File$, der Funktions-Parameter Length muss mit #Null angegeben werden
#XUP_FIMPEntpacken von Amiga FileImploder & #XUP_XPKF IMPL/ATN!
#XUP_GZIPEntpacken von (G)Zip & #XUP_XPKF GZIP
#XUP_JCG1Entpacken von JCalG1 - nur für x86
#XUP_LZMAEntpacken von LZMA
#XUP_NONEEntpacken von #XUP_XPKF NONE
#XUP_PACKBereitstellung der Packfunktionen für APLB, BLZ1, BZP2, FIMP, GZIP, JCG1 (x86), LZMA
#XUP_PP20Entpacken von Amiga PowerPacker Data & #XUP_XPKF PWPK
#XUP_PUREAlternative Verwendung der nativen Purebasic Static-Libs BLZ1, BZP2, GZIP, JCalG1 (x86)
#XUP_PX20Entkodieren von Amiga PowerPacker verschlüsselten Daten
#XUP_SQSHEntpacken von #XUP_XPKF SQSH
#XUP_UICEEntpacken von Atari Atomik 3.5 / Ice! 2.1 + 2.4
#XUP_XPKFVerwendung diverser Amiga XPK Compressor (PWPK 0.90, GZIP, BZP2, IMPL, SQSH, NONE) - IMPL enpackt fehlerhaft wenn Originaldatei größer 65535 Bytes ist (Custom Header) - NONE dient nur zu Testzwecken, ebenfalls fehlerhaft (Header unstimmig mit Dateigröße).
