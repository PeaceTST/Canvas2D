C2D::Init()

Syntax

C2D::Init(#Gadget, Speed=5, Color.l=#Black)

Beschreibung

Initialisiert ein zuvor definiertes CanvasGadget für weitere grafische Darstellungen.

Wichtig : Dieser Befehl muss vor allen anderen Anweisungen als erstes aufgerufen werden.

Parameter

 #Gadget  Nummer des zuvor erstellten CanvasGadget
 Speed  Wartezeit in Millisekunden für das Neuzeichnen. Standard ist 5 Millisekunden.
 Color.l  Hintergrundfarbe. Standard ist Schwarz.

Rückgabewert

Diese Funktion gibt keinen Wert zurück.

Anmerkung

Erst nach C2D::Init() können weitere Canvas 2D Funktionen aufgerufen werden.

Beispiel

XIncludeFile "..\Include\C2D_Module.pbi"

#SCR_Z = 1.25
#SCR_W = 512 * #SCR_Z
#SCR_H = #SCR_W * 0.6

#G_C2D = 0

OpenWindow(0, 0, 0, #SCR_W, #SCR_H, "Font/Text", #PB_Window_SystemMenu|#PB_Window_ScreenCentered)
CanvasGadget(#G_C2D, 0, 0, #SCR_W, #SCR_H) : DisableGadget(#G_C2D, 1)

C2D::Init(#G_C2D)

C2D::FontRawInit(0)
C2D::FontZoom(0, 16 * #SCR_Z, 24 * #SCR_Z)
C2D::FontColor(0, $0FB9FF)

C2D::TextSinusInit(0, 0, PeekS(?l_Text), 15, 80 * #SCR_Z)

y = (#SCR_H - C2D::TextH(0)) * 0.5

Repeat
 Select WindowEvent()
  Case #Null
   If C2D::Start()
    C2D::TextSinusDraw(0, y, 0.8 * #SCR_Z)
    C2D::Stop()
   Else
    Delay(1)
   EndIf
  Case #PB_Event_CloseWindow
   Break
 EndSelect
ForEver

C2D::Free()

DataSection
 l_Text: : Data.s "THIS IS AN TEXT-SCROLLER EXAMPLE BY CANVAS 2D"
EndDataSection
