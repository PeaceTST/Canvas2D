Macros
Kurze Übersicht und Beschreibung

Macro Name Beschreibung
C2D::MA_Time()Rückgabewert entspricht der aktuellen Zeit, diese ist in C2D::C2D\Time zwischengespeichert - s. #IsC2D_Time
C2D::MA_ERROR(TEXT)Dient zur Fehlerermittlung während dem Kompilieren. Sollte ein Fehler auftreten, wird entsprechender TEXT angezeigt und der Kompiliervorgang abgebrochen.
C2D::MA_RMP(N)Rückgabewert entspricht einem Zufallswert ±N, Beispiel:
n = C2D::MA_RMP(100) ermittelt einen Zufallswert zwischen -100 bis +100
MA_GSin(ANGLE)Ermittelt einen vordefinierten Sinuswert von ANGLE (Float)
MA_GCos(ANGLE)Ermittelt einen vordefinierten Cosinuswert von ANGLE (Float)
MA_GSinI(ANGLE)Ermittelt einen vordefinierten Sinuswert von ANGLE (Integer)
MA_GCosI(ANGLE)Ermittelt einen vordefinierten Cosinuswert von ANGLE (Integer)
MA_XC2D()Enthält die Konstante C2D::#C2D_VER$ welche die aktuelle Version von Canvas2D darstellt. Dient u.a. zum einfachen einbinden in Datasektionen
MA_XOS()Enthält die Konstante C2D::#C2D_XOS$ welche den Prozessor (x86 oder x64) während dem Kompilieren darstellt. Dient u.a. zum einfachen einbinden in Datasektionen.
MA_XPB()Enthält die Konstante C2D::#C2D_XPB$ welche die Purebasic Version während dem Kompilieren darstellt. Diese wird manuell ermittelt und reicht von "5.11" bis "5.73", Standart ist "5.40+" - für zukünftigen Versionen ggf. anpassen (Include\C2D_Enums.pbi).
MA_C2DOS(TITLE)Dient für einen Fenstertitel - Rückgabe ist folgende Zeichenkette, Beispiel:
MA_C2DOS("Mein Progamm") entspricht "C2D::Mein Programm (x86)"
MA_C2DPB(TITLE)Dient für einen erweiterten Fenstertitel - Rückgabe ist folgende Zeichenkette, Beispiel:
MA_C2DPB("Hallo") entspricht "C2D::Hallo / PB 5.70 (x64)"
