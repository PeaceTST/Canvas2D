#IsC2D_Stars3D = 13D Sternenhimmel
Stars3DColor(Color.l)Color = RGB-Farbwert für Standard-Sterne.
Stars3DDistance(z)Über z = ±255 wird die Entfernung der Sterne bestimmt. Je niedriger der Wert desto länger benötigt ein Stern bis er sichtbar ist.
Stars3DDraw(x.f=0, y.f=0)Zeichnet das Sternenfeld an Position x und y ausgehend vom Mittelpunkt des Canvas-Gadgets.
Stars3DFree()Löscht das Sternenfeld.
Stars3DInit(Number, Size, x, y, w, h, Speed.f, Image=#PB_Default)Initialisiert das 3D Sternenfeld. Number = #Anzahl der Sterne, Size = #Pixel-Größe des einzelnen Stern, x, y, w, h = Position sowie Breite und Höhe des Sternenfeld, Speed = Geschwindigkeit, Image = Standard oder #Bildnummer eines zuvor initialisierten Bildes - Beispiel: C2D::BitmapID(#BILD).
Stars3DSpread(Spread)Spread = #Streuungsfaktor der Sterne. Je höher die Wertangabe, desto weitverstreuter die Sterne.
#IsC2D_Stars3D = 2Beschleunigung
Stars3DInit(Number, Size, x, y, w, h, Speed.f, Image=#PB_Default)Initialisiert das 3D Sternenfeld mit deutlich schneller darstellbaren 1px Sternen. Die Parameter Size und Image werden hierbei nicht berücksichtigt.
