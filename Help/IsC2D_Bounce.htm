#IsC2D_Bounce = 1Vertikal beschleunigtes Auf-/Abprallen
IsBounce(ID)= Listeneintrag, #Null = Fehler
Bounce(ID)= Vertikale Position (Integer). Wiederholte Aufrufe mit gleicher ID sollten vermieden werden, da jeder Aufruf zur Erhöhung der Beschleunigung führt.
BounceFree(ID)Löscht einen- oder alle (ID = #PB_All) Bounce-Effekte.
BounceInit(ID, y, h, Acceleration)Initialisiert einen Bounce-Effekt. ID = Zuordnung, y = vertikale Startposition, h = Höhenbereich für den "freien" Fall, Acceleration = Beschleunigung 1/1000.
