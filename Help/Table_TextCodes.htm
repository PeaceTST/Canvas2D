Steuer-Codes für Bitmap-Font Textdarstellung
{Code,Param.f}

Code # Param.fIsC2D_(Effekt) = 2Beispiel
{Code,Param.f}		Effekt
#C2C_Bitmap0#BitmapID#IsC2D_PageText
#IsC2D_ScrollText		{0,3,(x,y)}
#BitmapID 3 an Textposition darstellen		Bitmap-Bild einfügen. #BitmapID = 0 entspricht {0,0} oder {0}. Wichtig: einzubindende Bitmap-Bilder müssen vor der Texterstellung initialisiert werden, #IsC2D_Bitmap muss daher aktiviert sein! Die zusätzlichen Parameter x,y dienen nur bei PageText als Positions-Offsets.
#C2C_Font1#FontID#IsC2D_PageText
#IsC2D_MoveText
#IsC2D_ScrollText		{1,2}Text
Text mit der #FontID 2 darstellen		Setzt einen neuen Font. #FontID = 0 entspricht {1,0} oder {1}. Wichtig: entsprechende Fonts müssen vor der Texterstellung initialisiert werden!
#C2C_Pause2Millisekunden#IsC2D_PageText
#IsC2D_ScrollText		Text{2,5000}
Text 5 sek. pausieren		Pausiert eine Seitendarstellung bzw. die Bewegung eines ScrollText. Eine Sekunde {2,1000} oder {2}.
#C2C_Space3%Prozent#IsC2D_ScrollTextText{3,25}Text
Textabstand von 25% einfügen		Setzt einen prozentualen Abstand zum nächsten Buchstaben. Die Prozentwerte beziehen sich auf die Breite des Canvas. Ganze Canvas-Breite {3,100} oder {3}.
#C2C_Speed4Geschwindigkeit#IsC2D_ScrollText{4,3}Text
Text mit 3facher Geschwindigkeit bewegen		Ändert die Geschwindigkeit. Standard {4,1} oder {4}.
#C2C_Center5%Prozent#IsC2D_ScrollText{5}Text{5}{2}
Text zentriert darstellen und für 1 sek. pausieren		Zentriert einen Text welcher innerhalb von {5}Text{5} angegeben werden muss. Optional kann für das abschließende {5} mittels Param.f bezogen auf die prozentuale Canvas-Breite eine zusätzliche Positionskorrektur erfolgen. Ein zentrierter Text kann anschließend mit #FC_Pause ohne weitere Freizeichen pausiert werden.
#C2C_Return6Rückgabewert#IsC2D_ScrollText{6,255}Text
Rückgabewert an Textposition beträgt 255		Bestimmt den Rückgabewert an Textposition für das Hauptprogramm. Dieser Wert kann für einen positionsgenauen Programmablauf eingesetzt werden.
#C2C_MoveY7%Prozent#IsC2D_ScrollText{7,-10}Text
Verschiebt nachfolgenden Text um 10% nach oben		Verschiebt einen Text ausgehend von aktuell vertikaler Position. Die Prozentangaben beziehen sich auf die Höhe des Canvas. Standard {7,0} oder {7}.
#C2C_Pixel3D8#Pixel3DID#IsC2D_ScrollText{8,0}Text
#Pixel3dID 0 an Textposition darstellen		Pixel3D-Bild einfügen. #Pixel3dID = 0 entspricht {8,0} oder {8}. Wichtig: einzubindende Pixel3D-Bilder müssen vor der Texterstellung initialisiert werden, #IsC2D_Bitmap muss daher aktiviert sein!
