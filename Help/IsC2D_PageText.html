#IsC2D_PageText = 1Text seitenweise anzeigen
PageTextColor(Color.l)Hintergrundfarbe der Seiten
PageTextDraw(x.f=0, y.f=0)= #Aktuelle Seite (0 … n). Kontinuierliches anzeigen der Textseiten. Die Parameter x/y dienen als Offset zur Positionskorrektur innerhalb der Seite.
PageTextEffect(Mode, Effect, Speed.f, Delay.f)Darstellungseffekt zum Anzeigen / Ausblenden einer Textseite. Mode = 1 (Seite anzeigen), Mode = 0 (Seitenwechsel), Effect = Ein-/Ausblendeffekt, Speed = #Geschwindigkeit pro Buchstabe, Delay = #Wartezeit bis zum nächsten Buchstaben.
PageTextFree()Löscht den PageText.
PageTextID(ID=0)= #Aktuelle Seite. Sollte ID > 0 übergeben werden, so wird die nächste Seitennummer entsprechend ID angezeigt.
PageTextInit(*Memory.Ascii, x, y, w, h, Flags=#Null)Initialisiert den Seitentext welcher in *Memory vorliegt. x, y = Position auf dem Canvas sowie w, h = Breite und Höhe einer Seite. Über Flags kann die Ausrichtung der Textzeilen festgelegt werden. #Tab wird unterstützt. #LF oder | bedeutet neue Textzeile innerhalb einer Seite (Zeilenumbruch). Das Zeichen ~ oder #Page_Next erstellt eine neue Seite mit nachfolgenden Textzeilen.
PageTextWait(Time, PageID=0)Time = Wartezeit in Millisekunden bis zum Seitenwechsel. Die Standardeinstellung beträgt 5000ms (5 Sekunden). Wird der Parameter PageID von 1..n Seite angegeben, wird nur die Wartezeit der angegebenen Seite eingestellt. Eine Zeitangabe von -1 wird die entsprechende Seite nicht anzeigen. PageTextWait() sollte stets vor PageTextInit() aufgerufen werden, da textinterne Zeiteinstellungen mittels "{2,#}" verändert werden.
#IsC2D_PageText = 2Erweiterung für Text-Effekte (s. Tabelle)
PageTextInit(*Memory.Ascii, x, y, w, h, Flags=#Null)Initialisiert den Seitentext welcher in *Memory vorliegt. Neben dem Sonderzeichen #Tab werden zusätzlich innerhalb des zu initialisierenden Text integrierte Steuer-Codes unterstützt mit der Syntax "{Code,Param.f}". Eine Übersicht der Steuer-Codes wird in Tabelle aufgeführt. Derzeit werden die Parameter #C2C_Bitmap {0,#}, #C2C_Font {1,#} und #C2C_Pause {2,#} unterstützt.

PageTextFxChange(FX)Privat, ändert den Darstellungseffekt.
