#IsC2D_Ball3D = 1Ball3D Objekt
IsBall3D(ID)= Listeneintrag, #Null = Fehler
Ball3DAngle(ID, x.f, y.f, z.f)Blickwinkel auf ein Objekt.
Ball3DCatchTheme(*Memory.Integer)Ball-Grafik für Objekte bereitstellen. *Memory = Tabelle mit Zeigern auf Grafikdaten (s. Tabelle).
Ball3DCount(ID)= #Anzahl der tatsächlich erstellten Bob-Bilder.
Ball3DDefaultTheme(*Memory.Long=0)Erstellt BallBobs in Standardfarben (*Memory=0) oder eigener Färbung (*Memory=@Color.l[8])
Ball3DDraw(ID, x.f=0, y.f=0, Alpha=255, Fade=0)= Darstellung eines Ball3D-Objekt ausgehend vom Mittelpunkt des Canvas, Fade = Tiefenwirkung über Nebelsimulation (0 = aus … 255 = dicht). Rückgabewert = #Anzahl der sichtbaren BallBobs.
Ball3DFree(ID)Löscht ein- oder alle Objekte (ID = #PB_All)
Ball3DImage(BallID, Image=-1)= #Nummer des Ball-Bildes (ID = 0 … 7), #Null = Fehler. Um ein Ball-Bild mit einem Bild auszutauschen muss bei Image eine gültige Bildnummer übergeben werden.
Ball3DInit(ID, *Memory, Size, Gap.f=1.0)= Erstellt ein Objekt aus den Daten in *Memory oder aus @Dateiname (#IsC2D_File muss hierzu aktiviert sein). Size = Größe des Objekts (1.0 ≙ 100%), Gab = prozentualer Abstand zwischen einzelnen Bobs (1.0 ≙ 100% = Standard). Rückgabewert = #Anzahl der erstellten Bob-Bilder.
Ball3DRotate(ID, x.f, y.f, z.f)Rotiert ein Objekt.
Ball3DSpin(ID, Factor.f, BallID=#PB_All)Eigenrotation eines- oder aller BallBobs eines Objekt.
Ball3DLoadTheme(Path$, Format$="png")#IsC2D_File muss aktiviert sein! Lädt Bilder für Ball-Grafiken, Format$ = Bildformat ("bmp","png"…). Dateinamen müssen entsprechend der Angabe Format$ = "png" in der Datei-Bezeichnung: 0.png … 7.png vorliegen (s. Tabelle).
#IsC2D_Ball3D = 2Erweiterung
Ball3DAnim(ID, AnimID, BallID)Ersetzt eine BallID nit einer zuvor initialisierten Animation (#IsC2D_Anim muss aktiviert sein)
Ball3DExplode(ID, Factor)Explosion eines Objekt. Factor = 1 oder 2 für Streuungs-Bewegung bis zur erneuten Zusammensetzung.
Ball3DMerge(ID1, ID2)Vereint das Objekt ID2 in ID1.
Ball3DStars(ID, Number, Radius, Size, BallID=3, Rnd.f=0)Erstellt ein statisches 3D-Sternenfeld, Rnd = Streuungsfaktor.
