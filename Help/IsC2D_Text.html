#IsC2D_Text = 1Bitmap Text
IsText(ID)= Listeneintrag, #Null = Fehler
TextDraw(ID, x.f, y.f, Alpha=255, Flags=#Null)Zeichnet einen zuvor initialisierten Textblock an Position x,y. Alpha = Transparenz (0 … 255). Wird mit Flags = #C2F_Center/X/Y oder #C2F_Bottom/Right angegeben, dienen x,y als zusätzliche Offset-Korrektur. Über FontSelect() wird zuvor der zu verwendete Bitmap-Font festgelegt.
TextFree(ID)Löscht ein- oder alle initialisierten Texte (ID = #PB_All)
TextH(ID)= #Pixel-Höhe
TextInit(ID, *Memory.Ascii, Flags=#Null)Initialisiert einen Text mit der Kennung ID welcher in *Memory vorliegt. Mit Flags = #C2F_CenterX wird der Text zeilenweise zentriert dargestellt.
TextStringDraw(x.f, y.f, Text$, Alpha=255)Zeichnet einen variablen Text. Entgegen TextDraw() muss dieser nicht zuvor initialisiert werden, ist jedoch in seiner Darstellung zeitaufwendiger. Vor jeder Darstellung kann mittels FontSelect() ein beliebiger Bitmap-Font erneut zugeordnet werden.
