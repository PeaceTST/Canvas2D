#IsC2D_Anim = 1Animation
AnimCopper(ID, *Memory.Long, Flags=0)Belegt alle Frames mit einem Copper-Farbverlauf.
AnimCopy(ID, NewID, Flags=0)Erstellt eine Kopie der Animation ID nach NewID. Flags = 0 beide Animationen verwenden die gleichen Frames um Speicher zu sparen. Flags = 1 es werden separate Frames erstellt für individuelle Größenveränderungen. Wird für NewID = #PB_Any übergeben, ist der Rückgabewert die neue ID.
AnimCount(ID)= Ermittelt die Frame-Anzahl der Animation ID. Die Zählung ist beginnend bei 0 (n-1).
AnimDelay(ID, Time, FrameID=#PB_All)Wiedergabegeschwindigkeit, Time = Dauer in Millisekunden. Über FrameID kann die Anzeigedauer eines einzelnen Frame festgelegt werden.
AnimDirection(ID, Frames)Vor- oder Rückwärts abspielen, Frames bestimmt die Schrittanzahl (Beispiel: 3 = nur jedes dritte Frame wird angezeigt, -2 = nur jedes zweite Frame wird von rückwärts angezeigt)
AnimDraw(ID, x.f, y.f, Alpha=255, Flags=0)= Darstellung einer Animation. Rückgabewert = #Aktuelles Frame
AnimFlip(ID, Flags=0)Spiegelt alle Frames einer Animation horizontal (Flags = 0) oder vertikal (Flags = #C2DF_Vertical)
AnimFrame(ID, FrameID=#PB_Default)= Setzt das aktuelle Frame-Bild bzw. #PB_Default = ermittelt das aktuelle #Frame
AnimFree(ID)Löscht eine- oder alle Animationen (ID = #PB_All)
AnimH(ID)= #Pixel-Höhe
AnimInit(ID, Image, x, y, Frames=#PB_All)Intialisiert eine Animation. x und y bestimmen die Anzahl horizontaler (x) und vertikaler (y) Frame-Bilder. Über Frames kann die maximale Frame-Anzahl variiert werden. Wird für die Initialisierung #PB_Any übergeben, ist der Rückgabewert die neue ID.
AnimPause(ID, State=1)Pausieren (1) oder Pause beenden (0)
AnimPingPong(ID, Mode=1)Am ende Animation rückwärts abspielen (ping-pong). Mode = 0 schaltet die Option wieder aus.
AnimPlayDraw(ID, x.f, y.f, w=#PB_Default, h=#PB_Default)= Wiederholungen bis ende. Spielt eine Animation an Position x, y mit der Pixelgröße w, h ab. Die Wiederholung entspricht dem Parameter Count bei AnimPlayStart().
AnimPlayStart(ID, Count=1)Initiert das Abspielen einer Animation. Der Parameter Count bestimmt die Anzahl der Wiederholungen. Nach "Count" Wiederholungen wird die Animation nicht mehr über AnimPlayDraw() dargestellt.
AnimRange(ID, FrameID, Number)Legt das FrameID=Startframe und die Number=Anzahl der wiederzugebenden Frames fest.
AnimScale(ID, Ratio.f)Proportionale Größenveränderung (Ratio = 1.0 ≙ 100%)
AnimSeen(ID)= #True wenn sich die Animation mit der Kennung ID im sichtbaren Bereich des Canvas befindet.
AnimW(ID)= #Pixel-Breite
AnimX(ID)= Horizontale Position.f der Animation mit der Kennung ID.
AnimY(ID)= Vertikale Position.f der Animation.
AnimZone(ID, x.f, y.f)= #True wenn sich die Position von x / y (z.B. Maus-Koordinaten) innerhalb der Animation mit der Kennung ID befindet.
AnimZoom(ID, w=#PB_Default, h=#PB_Default)Größenveränderung in Pixel.
IsAnim(ID)= Listeneintrag, #Null = Fehler
