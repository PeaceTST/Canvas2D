#IsC2D_Pixel3D = 1Pixel3D Bitmap
IsPixel3D(ID)= Listeneintrag, #Null = Fehler
Pixel3DSAngle(ID, x.f, y.f, z.f)Blickwinkel auf die Bitmap.
Pixel3DAxis(ID, x.f, y.f)Verschiebt die Achsen einer Pixel3D-Bitmap relativ zum Mittelpunkt.
Pixel3DClip(ID, x, y, w, h)Bestimmt die Begrenzungen für die sichtbare Darstellung.
Pixel3DCount(ID)= Anzahl der verwendeten Farbpixel einer Bitmap.
Pixel3DDraw(ID, x.f=0, y.f=0, Alpha=255)= Anzahl der dargestellten Pixel, eine Pixel3D-Bitmap wird an Position x/y vom mittleren Null-Punkt aus dargestellt. Über den Parameter Alpha kann die Transparenz bestimmt werden.
Pixel3DDrawColor(ID, x.f=0, y.f=0, Color.l)= Anzahl der dargestellten Pixel, eine Pixel3D-Bitmap wird an Position x/y vom mittleren Null-Punkt aus dargestellt. Der Parameter Color muss einen Farbwert enthalten in welchen das Bild dargestellt wird. Der Alphakanal wird unterstützt.
Pixel3DFree(ID)Löscht eine- oder alle (ID = #PB_All) Pixel3D-Bitmaps.
Pixel3DH(ID)= entspricht die Hälfte der tatsächlichen Pixelhöhe vom mittleren Null-Punkt.
Pixel3DInit(ID, Image, Color.l=#Black)Initialisiert ein Pixel3D-Bild mit der Kennung ID welches aus einer zuvor initialisierten Image (#Bildnummer) erstellt wird. Über Color wird ein transparenter Farbwert bestimmt, Standard ist Schwarz = Transparent. Es ist zu beachten, daß nicht die Größe/Breite des Bildes die Darstellungsgeschwindigkeit beeinflußt, maßgeblich ist die Anzahl der Farb-Pixel. Rückgabewert = Anzahl der tatsächlich verwendeten Pixel. Transparente Pixel bzw. Farbpixel mit dem Wert Color werden aus Gründen der Geschwindigkeit nicht berücksichtigt.
Pixel3DRotate(ID, x.f, y.f, z.f)Rotiert eine Bitmap um die Achsen der jeweiligen Geschwindigkeitsangaben.
Pixel3DW(ID)= entspricht die Hälfte der tatsächlichen Pixelbreite vom mittleren Null-Punkt.
