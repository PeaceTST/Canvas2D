#IsC2D_File = 1Datei
FileLoad(File$, Flags=0)= *Datenspeicher. Einlesen einer Datei in einen lokalen Zwischenspeicher. Dieser Zwischenspeicher wird standardmäßig bei erneuten Aufruf gelöscht (Flags=0), somit sollte eine direkte Verarbeitung durchgeführt werden. Ist Flags=1 wird keine Löschung bei erneuten Aufruf durchgeführt. Wurde #IsC2D_Network aktiviert, werden Dateien (HTTP://) auch aus dem Internet unterstützt.
FilePath(Path$)Setzt den Standardpfad für Dateizugriffe, ähnlich dem nativen IncludePath, Online-Pfade werden unterstützt.
#IsC2D_File = 2Pfad
FileParent(Path$=#Null$, Dirs=1)= Pfad des Elternverzeichnis ausgehend von Path$ (#Null$ = aktuelles Verzeichnis). Dirs = Anzahl der Elternverzeichnisse.
FileSave(File$, *Memory, Length)= Tatsächlich gespeicherte Bytes. Speichert den Dateninhalt von *Memory mit der Größe Length in die Datei File$.
