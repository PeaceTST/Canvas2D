#IsC2D_Line3D = 1Line3D Objekt
IsLine3D(ID)= Listeneintrag, #Null = Fehler
Line3DAngle(ID, x.f, y.f, z.f)Blickwinkel auf das Objekt.
Line3DColor(ID, Color.l)Linienfarbe, derzeit kann ein Objekt nur aus 1 Farbe bestehen. ID = #PB_All ändert die Linienfarbe für alle Objekte. Die Standardfarbe ist rot.
Line3DDepth(ID)Pixel-Tiefe eines Objekt
Line3DDraw(ID, x.f=0, y.f=0, zx.f=1.0, zy.f=1.0)Zeichnet ein Objekt an Position x/y. Über die Parameter zx und zy wird eine Größenveränderung durchgeführt (1.0 ≙ 100%)
Line3DFog(ID, Fog.f)Simuliert eine Tiefenwirkung durch scheinbaren Nebeleffekt (entferntere Linien werden transparenter dargestellt). ID = #PB_All wendet den Effekt auf alle Objekte an.
Line3DFree(ID)Löscht ein- oder alle Line3D-Objekte (ID = #PB_All)
Line3DHeight(ID)Pixelhöhe eines Objekt.
Line3DInit(ID, *Memory.RS_Line3D, Ratio.f=1.0, z.f=1.0)Initialisiert ein Objekt mit der Kennung ID welches in *Memory vorliegt oder als @Dateiname übergeben wird (#IsC2D_File muss hierzu aktiviert sein). Über Ratio kann eine proportionale Größenangabe bestimmt werden (1.0 ≙ 100%). Mittels Parameter z wird die Tiefe eines Objekt variiert (1.0 ≙ 100%).
Line3DRotate(ID, x.f, y.f, z.f)Rotiert ein Objekt um die Achsen der jeweiligen Wertangaben.
Line3DSquare(ID, Size.f, Ratio.f=0)Faktor für das Eckmaß eines L3D-Objekts.
Line3DWidth(ID)Pixelbreite eines Objekt.
#IsC2D_Line3D = 2Erweiterung
Line3DBuild(ID, Mode, Speed)Die Objektdarstellung wird Linie für Linie aufgebaut (Mode = 1) oder abgebaut (Mode = -1) in der Geschwindigkeit Speed (Millisekunden) / s. Line3DIsBuild()
Line3DFade(ID, Speed)Blendet ein Objekt ein oder aus in Millisekunden (Speed > 0 = einblenden, Speed < 0 ausblenden) / s. Line3DIsFade()
Line3DIsBuild(ID)= #Null wenn ein Objekt vollständig auf- / abgebaut wurde.
Line3DIsFade(ID)= #Null wenn ein Objekt vollständig ein- / ausgeblendet wurde.
