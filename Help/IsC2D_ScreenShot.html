#IsC2D_ScreenShot = 1Canvas-Ausgabe in das Clipboard kopieren
ScreenShot()Erstellt ein Bildschirmfoto der aktuellen Canvas-Ausgabe und kopiert dieses in das Clipboard.

; Beispiel
Repeat
   Select   WindowEvent()
      Case  #Null :  Delay(1)
      Case  #WM_KEYDOWN
         Select   EventwParam()
            Case  #VK_F1      :  C2D::ScreenShot()
            Case  #VK_ESCAPE  :  Break
         EndSelect
   EndSelect
ForEver
