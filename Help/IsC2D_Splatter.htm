#IsC2D_Splatter = 1Splatter
SplatterAcceleration(ID, x, y)Beschleunigung der Splatter-Partikel in den Richtungen x und y
SplatterAlpha(ID, Alpha=$FF)Alpha = Transparenz der Partikel von 1 bis $FF
SplatterBounce(ID, Percent.f)Percent = die prozentuale Springhöhe nach dem Aufprallen auf den Boden.
SplatterDraw(ID, h.f)Darstellung des Splatter mit der Kennung ID. h = die Fallhöhe ausgehend von SplatterStart(). Rückgabewert = die Anzahl der aktiven Splatter-Partikel.
SplatterEnergy(ID, Energy)Maximale Leuchtdauer der Splatter-Partikel bevor die Ausblendung beginnt.
SplatterFade(ID, Fade)Geschwindigkeit mit der die Partikel nach beenden der Leuchtdauer ausgeblendet werden. Fade muss zwischen 1 (langsam) und 255 (schnell) angegeben werden.
SplatterFree(ID)Löscht einen- oder alle aktiven Splatter (ID = #PB_All)
SplatterGravity(ID, Gravity)Bestimmt die Gravitation welche die Splatter-Partikel in ihrer vertikalen Fallgeschwindigkeit beeinflussen. Je höher die Gravitation, desto schneller fallen die Partikel zu boden.
SplatterH(ID)= Pixelhöhe der Partikel.
SplatterImage(ID, Image)Setzt ein Bild für jeweilige Splatter-Partikel, Image = Bildnummer.
SplatterInit(ID, Size, Number, Color.l=#Red)Intialisiert einen Splatter. Size = Pixelgröße der jeweiligen Partikel. Number = Anzahl der Partikel eines Splatter. Color = RGB-Farbwert
SplatterLoop(ID, State)Splatter in Endlosschleife darstellen. State = 1 schaltet den Endlosmodus an, State = 0 wieder aus.
SplatterScale(ID, Ratio.f)Proportionale Größenveränderung (Ratio = 1.0 ≙ 100%)
SplatterSpread(ID, SpreadX, SpreadY)Streuungs-Breite / Höhe vom Startpunkt der Partikel aus.
SplatterStart(ID, x.f, y.f)Initiert den Splatter mit der Kennung ID an Position x / y. Mittels SplatterDraw() werden die Partikel dargestellt.
SplatterStop(ID=#PB_All)Beendet die Darstellung des Splatter mit der Kennung ID. ID = #PB_All beendet alle darestellten Splatter.
SplatterW(ID)= Pixel-Breite der Partikel.
SplatterAnim(ID, AnimID))Verwendet eine zuvor initialisierte Animation mit der Kennung AnimID. Jeder Partikel wird als Animation dargestellt.
IsSplatter(ID)= Listeneintrag, #Null = Fehler
