#IsC2D_Bitmap = 1Bilder für Animation, Font etc.
BitmapAdd(ID, Image)= #Bildnummer, fügt ein Bild zur Bitmap-Liste hinzu.
BitmapDraw(ID, x.f, y.f, Alpha=255, Flags=0)Darstellung einer Bitmap im CanvasGadget. Flags = #C2F_Right [Bottom | Center | CenterX | CenterY]
BitmapFlip(ID, Flags=0)Spiegelt eine Bitmap horizontal (Flags = 0) oder vertikal (Flags = #C2F_Vertical).
BitmapFree(ID)Löscht ein- oder alle Bitmap-Bilder (ID = #PB_All)
BitmapH(ID)= #Pixel-Höhe
BitmapID(ID)= *Handle der Bitmap (ImageID)
BitmapImage(ID)= #Nummer der Bitmap.
BitmapInit(ID, *Memory, Length=0, Color.l=#PB_Default)= #Bildnummer, initialisiert ein Bitmap-Bild mit der Kennung ID welches in *Memory vorliegt, #PB_Any wird unterstützt. Length = die Größe des *Memory oder Ende des Speicherbereich. Sollte für Length = 0 angegeben werden, wird *Memory als Zeiger auf einen @Dateinamen interpretiert, hierzu muss #IsC2D_File aktiv sein. Color = transparente Farbe.
BitmapRange(ID, x.f, y.f)= #True wenn die Koordinaten von x / y (z.B. Maus-Koordinaten) innerhalb der dargestellten Bitmap mit der Kennung ID befindet.
BitmapScale(ID, Ratio.f)Proportionale Größenveränderung (Ratio = 1.0 ≙ 100%)
BitmapW(ID)= #Pixel-Breite
BitmapX(ID)= Horizontale Position.f der dargestellten Bitmap mit der Kennung ID.
BitmapY(ID)= Vertikale Position.f der dargestellten Bitmap.
BitmapZoom(ID, w=#PB_Default, h=#PB_Default)Größenveränderung in Pixel.
IsBitmap(ID)= Listeneintrag, #Null = Fehler
#IsC2D_Bitmap = 2Erweiterung
BitmapLoop(ID, Flags)Erstellt aus der Bitmap mit der Kennung ID ein Endlos-Bild. Flags = #F_Horizontal für horizontale Anpassung und/oder #F_Vertical für vertikale Anpassung. Wichtig: BitmapLoop() muss außerhalb von Start() / Stop() aufgerufen werden!
BitmapScroll(ID, x, y, Time=0)Verschiebt ein Bitmap-Bild um x/y Pixel in horizontaler- / vertikaler Richtung. Der Parameter Time verzögert die Scroll-Bewegung zur etwas genaueren Einstellung der Geschwindigkeit. Wichtig: BitmapScroll() muss außerhalb von Start() / Stop() aufgerufen werden!
PrivatIntern
BitmapCopy(ID)= #Bildnummer, erstellt eine Kopie von ID in 32 Bit-Farbtiefe.
