#IsC2D_Buffer = 1Direkt-Zugriff auf Canvas-Grafikspeicher
BufferBackDraw()Kopiert den internen Zwischenspeicher direkt in den Canvas-Grafikspeicher. Transparenz wird nicht unterstützt.
BufferBackGrab()Kopiert den Canvas-Grafikspeicher innerhalb von StartDrawing() / StopDrawing() in einen internen Zwischenspeicher.
BufferClear()Schnelles löschen des Canvas-Grafikspeicher mit der Farbe Schwarz.
BufferCloneY(y0, y1, h)Kopiert einen vertikalen Bereich von y0 nach y1 in der Pixel-Höhe h.
BufferFrontDraw(y=0, h=0)Schnelles kopieren des internen Zwischenspeicher direkt in den Canvas-Grafikspeicher. Der Farbwert #Black wird nicht gezeichnet (Transparent). Die Parameter y = vertikale Startposition sowie h = Höhe des einzufügenden Zwischenspeicher dienen zur Ressourcen-Schonung.
BufferFrontGrab()Kopiert den Canvas-Grafikspeicher innerhalb von StartDrawing() / StopDrawing() in einen internen Zwischenspeicher. Der Farbwert #Black ist Transparent.
BufferMirror(x, y, w, h, Mask=%11111111)Spiegelt einen Canvas-Bereich. Über Bit-Veränderung von Mask kann die Höhe und Zeilen-Spiegelung eingestellt werden.
BufferSinDraw(y=0)Transparentes zeichnen des internen Sinus-Speicher. Mittels dem Parameter y kann eine Darstellung beschleunigt werden, da hierbei ausgehend vom vertikalen Mittelpunkt die Zeichenopperation entsprechend der y Wertangabe beendet wird.
BufferSinX(x, y, w, h, Width, Frequency.f, Speed.f)Horizontale Wellenform.
BufferSinY(x, y, w, h, Height, Frequency.f, Speed.f, Flags=#True) Vertikale Wellenform. Mit Flags = 0 wird der Canvas-Grafikspeicher nach Bearbeitung in einen internen Sinus-Speicher abgelegt um diesen mit BufferSinDraw() darzustellen (Farbe Schwarz = Transparent)
#IsC2D_Buffer = 2Erweiterung
BufferBackScroll(x, y, Time=0)Endlose Verschiebung des BackBuffer in horizontale- / vertikale Richtung. Der Parameter Time verzögert die Scroll-Bewegung zur etwas genaueren Einstellung der Geschwindigkeit. Die horizontale Verschiebung (x) kann fehlerhaft sein, weshalb diese nur bedingt verwendbar ist!
BufferNoise(x, y, w, h, Noise=3, Color.l=$00FFFFFF)Simuliert ein TV-Rauschen, Noise = Intensität, Color = BGR Farbwert (kein Alpha)
PrivatIntern
BufferFree()Privat, wird bei Free() gelöscht.
BufferInit()Privat, wird bei Init() eingerichtet.
