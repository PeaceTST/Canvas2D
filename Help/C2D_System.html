Canvas2D Basis-Module

C2D ModuleBeschreibung
Color(Color.l)BGR-Hintergrundfarbe für das CanvasGadget.
Free()Löscht alle Effekte.
Init(Gadget, Speed=5, Color.l=#Black)Initialisiert C2D. Init() muss einmalig vor allen anderen C2D Effekten aufgerufen werden. Gadget = #CanvasGadet, Speed = #Wartezeit in Millisekunden bis zur Neuzeichnung, Color = Hintergrundfarbe.
Quality(Mode)Mode = 1 führt eine Größenänderung von Bitmaps ohne Kantenglättung durch ≙ #PB_Image_Raw. Mode = 0 Größenveränderung mit Kantenglättung ≙ #PB_Image_Smooth. Die nativen #Konstanten von Purebasic können für Mode übergeben werden.
Start()= #True wenn eine Zeichenausgabe nach Ablauf der Zeit Speed (s. Init()) ausgeführt werden kann.
Stop()Beendet die Zeichenausgabe. Eventuelle Anweisungen für Guru() und Help() werden ausgeführt.
Uni2Asc(*Text, Flags=#Null)= *Speicher welcher in *Text vorliegt wird von Unicode nach AscII konvertiert. Rückgabewert ist ein *Zwischenspeicher mit Textdaten in AscII (0-255 Zeichen) welcher bei erneutem Aufruf freigestellt wird, weshalb dieser direkt verwendet werden sollte. C2D unterstützt ausschließlich AscII, Purbasic verarbeitet in aktuellen Versionen nur noch Unicode weshalb aus Gründen der Kompatibilität eine Textanpassung durchgeführt wird. Flags = #F_Ucase ändert den Text in Großbuchstaben.
ViewPort(x.f=0, y.f=0)Zurücksetzen der Canvas-Zeichenausgabe. Nur sinnvoll bei Verwendung von Poly3D da hier die Ausgabe für jedes Polygon individuell eingestellt wird und Einfluss auf weitere Darstellungen ausübt. Über die Parameter x/y kann die horizontale- / vertikale Ausgabepositionen festgelegt werden.
