#IsC2D_SinusText = 1Bitmap Sinus-Text
IsSinusText(ID)= Listeneintrag, #Null = Fehler
SinusTextDraw(ID, y.f, Alpha=255)Zeichnet einen zuvor initialisierten Text mit der Kennung ID an zentrierter Position y. Alpha = Transparenz des Sinustext.
SinusTextFree(ID)Löscht ein- oder alle initialisierten SinusTexte (ID = #PB_All)
SinusTextInit(ID, *Memory.Ascii, Frequency.f=15.0, Height.f=70.0, YStep.f=5.0)Initialisiert einen Sinus-Text welcher in *Memory vorliegt. Frequency = Frequenz des Sinus, Height = Höhe des Sinus, YStep = Geschwindigkeit der Sinus-Frequenz.
SinusTextSpeed(ID, Speed.f)Speed = Bewegungsgeschwindigkeit von rechts nach links.
#IsC2D_SinusText = 2Erweiterung für Text-Effekte (s. Tabelle)
SinusTextInit(ID, *Memory.Ascii, Frequency.f=15.0, Height.f=70.0, YStep.f=5.0)Initialisiert einen SinusText welcher in *Memory vorliegt. Neben dem Sonderzeichen #Tab werden zusätzlich innerhalb des zu initialisierenden Text integrierte Steuer-Codes unterstützt mit der Syntax "{Code,Param.f}". Eine Übersicht der Steuer-Codes wird in Tabelle aufgeführt.
