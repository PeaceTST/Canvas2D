#IsC2D_Poly3D = 1Polygon mit beliebig vielen Eckpunkten
IsPoly3D(ID)= Listeneintrag, #Null = Fehler
Poly3DAngle(ID, x.f, y.f, z.f)Blickwinkel auf ein Polygon.
Poly3DBrush(ID, Image)Ein Bild (Image = #Bildnummer) wird als Füllung verwendet. Die Dimensionen beziehen sich auf die Begrenzungen welche in Poly3DClip() angegeben wurden. Kleine Bilder werden endlos gekachelt.
Poly3DClip(ID, x, y, w, h)Bestimmt die Begrenzungen für die sichtbare Darstellung. Wird für ID = #PB_All übergeben, werden die Begrenzungen für alle Polygone angewendet.
Poly3DColor(ID, Color.l, Pen.l=#PB_Ignore, w=0)Bestimmt einen neuen Farbwert für ein Polygon mit der Kennung ID. Color = RGB Farbe (kein Alpha), Pen = Umrandungsfarbe, w = Breite von Pen.
Poly3DDraw(ID, x.f=0, y.f=0)Ein Polygon wird an Position x/y vom mittleren Null-Punkt aus dargestellt. Positive x/y Werte verschieben ein Polygon nach rechts/runter, negative nach links/hoch.
Poly3DDrawLine(ID, x.f=0, y.f=0, Color.l)Ein Polygon wird an Position x/y vom mittleren Null-Punkt nur in seiner Umrandung (unausgefüllt) dargestellt. Color = RGB-Wert für die Umrandungsfarbe.
Poly3DFree(ID)Löscht ein- oder alle (ID = #PB_All) Polygone.
Poly3DInit(ID, *Memory.Union, Color.l=#White, Pen.l=#PB_Ignore)= Anzahl der Eckpunkte (Vertices), #Null = Fehler. Initialisiert ein Polygon mit der Kennung ID aus den Dateninhalt von *Memory. Color = Füllfarbe, Pen bestimmt bei Angabe eines Farbwert die Umrandungsfarbe. Ein jeweiliger Alpha-Kanal wird nicht ünterstützt. Die Initialisierung muss vor allen anderen Befehlen ausgeführt werden.
Poly3DRotate(ID, x.f, y.f, z.f)Rotiert ein Polygon um die Achsen der jeweiligen Geschwindigkeitsangaben.
Poly3DScale(ID, Ratio.f)Dimensioniert ein Polygon proportional, Beispiel Ratio = 2 entspricht doppelte Größe.
Poly3DZoom(ID, w.f, h.f)Vergrößert ein Polygon unproportional in w = Breite und h = Höhe.
