#IsC2D_MoveText = 1Text hoch- / runter rollen (Vor- / Abspann)
IsMoveText(ID)= Listeneintrag, #Null = Fehler
MoveTextColor(ID, Color.l)Hintergrundfarbe in Color = ABGR.
MoveTextDraw(ID, Alpha=255)Darstellung eines MoveText.
MoveTextFree(ID)Löscht ein- oder alle MoveText (ID = #PB_All)
MoveTextH(ID, Flags=#Null)= #Pixel-Höhe des gesamten Text (Flags = 1) oder #Pixel-Höhe des sichtbaren Text (Flags = 0)
MoveTextInit(ID, *Memory.Ascii, x, y, w, h, Flags=#Null)Initialisiert den MoveText welcher in *Memory vorliegt. x, y = Position auf dem Canvas sowie w, h = Breite und Höhe einer Seite. Über Flags kann die Ausrichtung der Textzeilen festgelegt werden. #Tab wird unterstützt. #LF oder | bedeutet ein Zeilenumbruch.
MoveTextSpeed(ID, Speed.f)Bewegungsgeschwindigkeit der Textdarstellung. Ist Speed > 0 wird der Text von oben nach unten rollen, Speed < 0 umgekehrt.
MoveTextW(ID)= #Pixel-Breite des sichtbaren MoveText.
MoveTextY(ID, y.f)Der Parameter y setzt die vertikale #Pixel-Position.
#IsC2D_MoveText = 2Erweiterung für Text-Effekte (s. Tabelle)
MoveTextInit(ID, *Memory.Ascii, x, y, w, h, Flags=#Null)Es werden zusätzlich innerhalb des zu initialisierenden Text Steuer-Codes mit der Syntax "{Code,Param.f}"unterstützt. Eine Übersicht der Steuer-Codes wird in Tabelle aufgeführt. Derzeit kann nur #FC_Font angewendet werden. Ein entsprechender Font sollte mit gleichen Attributen wie der Standartfont vorliegen.
