Hilfswerkzeugs

Alle Tools, zumindest die Selbst-Programmierten, entstanden nur- und nebenbei zur Entwicklungs-Erleichterung und sind keine Meilensteine in der Anwenderfreundlichkeit. Einige verstehe ich selbst nicht mehr so richtig, naja mal eben so ohne Anleitungserinnerung erstellt weil alles so "logisch" erschien ;)

Im Ordner Tools\ befinden sich einige Programme welche zur Erstellung von Ball3D-/Line3D-Objekte entstanden. Ebenso der bereits erwähnte AmpMaster-Installer sowie ein kleines Tool C2D_Compiler, mit welchem während der Testphase sämtliche Beispiele und Demos auf die Schnelle wiederholt kompiliert wurden (übrigens auch als ASM-Source oder UPX gepackt).

Im Ordner SCAL\ befinden sich die SCAL_x86.dll bzw. SCAL_x64.dll (MultiMedia Player Library) welche auf recht einfache Weise unterschiedlichste Tracker-/Musik-Formate (Tabelle) abspielen können (Netz- und Zip-gepackte Dateien werden übrigens auch unterstützt). Die jeweilige SCAL_x86-64.dll sollte sich im benannten Ordner bzw. im aktuellen Verzeichnis der ausführbaren Datei befinden.

Für den Fall, daß fast alle Effekte gleichzeitig angezeigt werden sollen, so kann dafür das Beispiel C2D_CanvasTest ausprobieren werden.

Der AmpPlayer benötigt die Installation von AmpMaster im Ordner AmpInstaller\.

Weiter befinden sich im Ordner Tools\ zusätzlich die Exe-Packer UPX und Petite welchen jeweils eine per Drag & Drop Batch-Datei beigefügt wurde. Einfach eine EXE/DLL Datei auf die BAT-Datei ziehen um die entsprechende Datei im bestmöglichen Verfahren zu komprimieren, dauert natürlich etwas. Übrigens evtl. auch nützlich um dieses nervig unbegründete alarmieren diverser AV-Tools zu umgehen, bei Bedarf austesten.

Dann gibt es noch das Tool V2M\conv2m.exe, dieses dient zum konvertieren von *.v2m Musikdateien falls diese Probleme beim abspielen machen.

Für Amiga-Anwender (WinUAE) wurde das Tool Amiga\Font2Raw (F2R) beigefügt welches Raw-Fonts erstellt. Zwei gepatchte Versionen (F2RLcase & F2RUcase) verwenden standardmäßig eine Zeichenkette wie sie in C2D für Fonts unterstützt wird.

Ansonsten, einfach mal Tools\ öffnen.
