#IsC2D_RotoZoom = 1Rotierende- & Größenverendernde Bild-Textur
RotoZoomAlpha(Alpha)Alpha = 0 bis 255 bestimmt die Transparenz der Textur.
RotoZoomBlend(Percent)Prozentuale Durchsichtigkeit der Textur. Bestehende Alphawerte werden berücksichtigt.
RotoZoomClip(x, y, w, h)Darstellungsbereich der rotierenden Textur.
RotoZoomSet(Time, Speed, Min, Max)Time = Wartezeit in Millisekunden für Rotation- / Größenveränderung. Speed = Schrittgröße in Pixel für Größenveränderung. Min / Max = minimale bzw. maximale Größendarstellung der Textur.
RotoZoomInit(Image, Size.f=256.0, Color.l=#PB_Ignore)Initialisierung eines Bildes zur Textur. Image = Bildnummer. Size = proportionale Pixelgröße der Textur, diese kann maximal 256 Pixel betragen. Color = Farbwert welcher als transparent berechnet- und nicht dargestellt wird.
RotoZoomDraw(rx.f, ry.f)Darstellung der Bild-Textur. rx / ry = horizontale- / vertikale Rotationsgeschwindigkeit.
