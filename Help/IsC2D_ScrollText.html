#IsC2D_ScrollText = 1Textzeile horizontal bewegen
IsScrollText(ID)= Listeneintrag, #Null = Fehler
ScrollTextDraw(ID, y.f, Alpha=255)= #F_End entspricht Textende. Zeichnet einen zuvor initialisierten ScrollText mit der Kennung ID an Position y. Alpha = Transparenz des Scrolltext.
ScrollTextFree(ID)Löscht einen- oder alle initialisierten ScrollTexte (ID = #PB_All)
ScrollTextH(ID)= #Pixel-Höhe
ScrollTextInit(ID, *Memory.Ascii)Initialisiert einen ScrollText welcher in *Memory vorliegt. Nur Sonderzeichen #Tab wird unterstützt.
ScrollTextSinus(ID, Height.f=0, Frequency.f=0, Speed.f=0)Startet / Stoppt die Sinusdarstellung. Height = Höhe des Sinus, Frequency = Frequenz des Sinus, Speed = Geschwindigkeit der Sinus-Frequenz. Wird für Height = 0 angegeben stoppt die Sinusdarstellung, Frequency und Speed werden erst durch neue Werte ≠ 0 verändert.
ScrollTextSpeed(ID, Speed.f)Speed = Bewegungsgeschwindigkeit von rechts nach links.
ScrollTextW(ID)= #Pixel-Breite
#IsC2D_ScrollText = 2Erweiterung für Text-Effekte (s. Tabelle)
ScrollTextDraw(ID, y.f, Alpha=255)= Param, entspricht Parameter für #FC_Return. Ein Parameter-Wert von -1 (#F_End) sollte nicht verwendet werden, da dieses für das Textende belegt ist.
ScrollTextInit(ID, *Memory.Ascii)Es werden zusätzlich innerhalb des zu initialisierenden Text Steuer-Codes unterstützt mit der Syntax "{Code,Param.f}". Eine Übersicht der Steuer-Codes wird in Tabelle aufgeführt.
ScrollTextPos(ID, Index=#Null)= Textposition. Ist Index > #Null wird entsprechend der Wertangabe die Position des ScrollText gesetzt. Dies kann in der Testphase unterstützend für das erstellen von Effekte {Code,Param.f} wirken um den TextScroller ab einer Index-Position zu starten.
