#IsC2D_NetWork = 1Unterstützung für Online-Dateien
NetWorkBytes(Length=4096)Length = Byte-Größe zur temporären Zwischenspeicherung eines Download. Höhere Werte beschleunigen den Download bei gleichzeitig höherer Beanspruchung der Systemressourcen.
NetWorkCallBack(*CallBack=0)Der Parameter *CallBack sollte bei Verwendung ein Zeiger auf eine @Procedure(Param) beinhalten. Über Param kann der prozentuale Status des Download ausgewertet werden. Sollte @Procedure(Param) einen Wert ≠ 0 zurückgeben, wird der Download abgebrochen.
NetWorkFree()Beendet die temporäre C2D/0.3 Internetverbindung.
NetWorkLoad(Url$)= *Datenspeicher, #Null = Fehler. Einlesen einer Online-Datei in einen lokalen Zwischenspeicher. Dieser Zwischenspeicher muss manuell freigegeben werden. Die Größe des *Datenspeicher kann nativ über MemorySize() ermittelt werden.
NetWorkSize(Url$)= Byte-Größe der Online-Datei Url$.
NetWorkSkip(Length=0)Length = Anzahl Bytes die von der Startposition einer Online-Datei übersprungen werden. Gedacht um individuelle Header diverser Pseudo-Formate zu überspringen. Wichtig: nicht erlaubte / vorgetäuschte Dateiformate können zu einer Sperrung durch den Hoster führen, das Risiko trägt alleine der Uploader!
#IsC2D_NetWork = 2Online-Datei herunterladen
NetWorkDownload(Url$, File$)= Gespeicherte Dateigröße, #Null = Fehler. Die Online-Datei Url$ wird aus dem Weltnetz heruntergeladen und in die Datei File$ gespeichert.
PrivatIntern
NetWorkInit(Url$)Öffnet eine temporäre C2D/0.3 Internetverbindung. Die Netzadresse Url$ wird in ein netzgültiges (canonicalized) Zeichenformat konvertiert.
NetWorkUrl(Url$)= Konvertierte Url$, unsichere Zeichen der Netzdatei werden vor Zugriff konvertert (Canonicalize, "#" = %23...)
